Slate Security

  • https://slatesecurity.ai
  • Overview
    • Slate is an agentic vulnerability scanner (DAST) built for teams in highly regulated and secure environments.
    • Scans web apps and APIs for security vulnerabilities, explains them in plain English, and ships AI-ready prompts to remediate any findings.
    • Finds vulnerabilties such as XSS, SQL injection, and more.
  • Design
    • Designed to be agent first, programmatic, and fully automatable.
    • An MCP server is available so AI agents can scan and pull findings directly in chat and remediate them.
    • API and CLI are available for programmatic access.
    • Web dashboard is available for a visual overview of findings.
    • Simple and easy enough for humans to manage, but powerful enough for AI agents to scan and remediate findings.
